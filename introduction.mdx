GLAM CLI is the official command-line tool for interacting with the GLAM Protocol on Solana. It provides a powerful interface for creating, managing, and operating GLAM vaults directly from your terminal.

## What is GLAM?

GLAM (Global Liquidity Asset Management) is a protocol on Solana that enables on-chain asset management through customizable vaults. Whether you're managing a simple portfolio or running a complex tokenized fund with performance fees and lock-up periods, GLAM provides the infrastructure you need.

## Key Features

### Vault Management
- Create and manage multiple vault types: standard vaults, tokenized vaults, and single-asset vaults
- Configure asset allowlists to control which tokens can be held
- Enable/disable vaults and manage ownership transfers
- View real-time token balances and holdings with USD valuations

### DeFi Integrations
GLAM CLI provides native support for major Solana DeFi protocols:
- **Jupiter**: Token swaps and limit orders
- **Kamino**: Lending, liquidity vaults, and yield farming
- **Drift**: Perpetual trading and vault strategies
- **Liquid Staking**: Support for various LST protocols including Marinade
- **CCTP**: Cross-chain transfers via Circle's CCTP protocol

### Tokenized Vaults
Create investment products with advanced features:
- Subscription and redemption management
- Performance and management fee structures
- Lock-up periods and minimum investment amounts
- Notify-and-settle mechanisms for controlled entry/exit
- Customizable token metadata (name, symbol, URI)

### Access Control
- Delegate management for multi-user access
- Protocol-specific permission management
- Integration allowlisting and configuration
- Emergency access controls

### Developer-Friendly
- JSON-based vault configuration
- Template files for quick setup
- Priority fee configuration for optimal transaction processing
- Staging environment support for testing

## Use Cases

**Portfolio Management**: Track and manage your Solana assets across multiple protocols with a unified interface.

**Tokenized Funds**: Launch on-chain investment funds with customizable fee structures and access controls.

**Trading Operations**: Execute swaps, perpetual trades, and manage liquidity positions across integrated protocols.

**Yield Optimization**: Deposit assets into lending protocols and liquidity vaults to earn yield on your holdings.
